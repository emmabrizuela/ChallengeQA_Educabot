## 2. Alcance
Se cubrirán pruebas funcionales sobre las funcionalidades mencionadas.

## 3. Requerimientos
- Navegador Google Chrome
- Conexión a internet

- Agregar y remover productos del carrito
- Checkout inicial

## 5. Estrategia de pruebas
